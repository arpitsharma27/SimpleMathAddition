This is simple Math Addition html.

**Parent**: 
Download the html file and show your kid how to open the html file. There is no internet required. 
The kid just answers the math questions. Once the kid is done hit END Game and then a download option will appear. 
This will be a report of what all questions your kid was asked and whether your kid got it correct or wrong.
It also tells you how much time your kid took to answer each question.
